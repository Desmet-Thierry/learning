Les Balises vues.

La balise <head>

La balise <head> fournis des informations sur la page, elle y intègre le titre de la page(de l'onglet) la description dela page, la langue, les feuilles de style etc.

La balise <body>

La balise <body> contient tout le contenu de la page qui sera afficher.

La balise <H1> (H2, H3, H4, H5, H6)

La balise <H1> (H2, H3, H4, H5, H6) representent les titres a différents degrès de la page.

La balise <p>

La balise <p> representent les paragraphes de la page.

La balise <img>

La balise <img> permet d'inserer une image dans la page.

La balise <table>

La balise <table> permet d'inserer une table dans la page.

La balise <tr>

La balise <tr> permet d'inserer une ligne dans la table.

La balise <td>

La balise <td> permet d'inserer une colonne dans la table.

La balise <ul>

La balise <ul> permet d'inserer une liste non ordonnee dans la page.

La balise <ol>

La balise <ol> permet d'inserer une liste ordonnee dans la page.

La balise <li>

La balise <li> permet d'inserer une liste dans la page.

La balise <a>

La balise <a> permet d'inserer un lien dans la page.
