Introduction

Les points clés a retenir sur l'html

Qu'est ce que l'html

l'html est un langage de balisage

Qu'est ce que le balisage

le balisage est le transfert de l'information d'un fichier texte vers un navigateur web

Pourquoi l'html ?

l'html est un language de balisage utilisée afin de mettre le contenue d'un site web dans un fichier html et de l'afficher le CSS servira plus tard a le mettre en forme

A quoi ressemble l'html


            //< = balise ouverture et > de fermeture ^^. 
            <!DOCTYPE html>
            <html lang="fr">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title>
            </head>
            <body>
                
            </body>
            </html>

Un des moyen d'avoir l'html dans le navigateur est la balise <xmp>

<h1>Ce contenu ne sera pas interprété</h1>

Une autre maniere d'avoir l'html dans le navigateur est la balise <pre> ou la balise <code> / <pre>
